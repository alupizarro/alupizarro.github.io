## Repositorios y Trabajos Destacados

### 1. Sistema-Reserva-Hotel-Diagrama-de-Clases-UML

Proyecto académico enfocado en el análisis y diseño de un sistema de gestión hotelera mediante diagramas UML. Incluye la identificación de clases, atributos, métodos y relaciones entre los distintos componentes del sistema, aplicando conceptos de programación orientada a objetos y modelado de software.

### 2. Emprendurismo-El-dilema-de-redes-sociales

Trabajo de investigación y análisis sobre el impacto de las redes sociales en la sociedad actual. Se abordan aspectos tecnológicos, éticos y sociales relacionados con el uso de plataformas digitales y su influencia en la comunicación y el comportamiento de los usuarios.

### 3. Sistema_de_reserva_de_hotel

Proyecto grupal de desarrollo de software orientado a la gestión de reservas hoteleras. Incluye funcionalidades vinculadas a la administración de clientes, habitaciones, reservas, estados, check-in y check-out, utilizando Git y GitHub como herramientas de trabajo colaborativo.

### 4. alupizarro.github.io

Sitio web personal desarrollado y publicado mediante GitHub Pages. Este proyecto funciona como portafolio académico y profesional, centralizando proyectos, conocimientos y experiencias adquiridas durante la formación en programación.
